# newsletter_php
---
#Подписка на новости сайта с рассылкой на электронную почту
---
Этапы:
1) Занесение электронных адресов подписчиков в базу данных;
2) Сбор данных о новости для письма и непосредственно рассылка по адресам;
---
Подробнее.
1 этап. Создание всплывающего окна на сайте (файл modal.html) с предложением о подписке на новости. Скрипт на JS (файл script.js), осуществляющий "всплытие" окна и его закрытие по нажатию пользователем на "крестик". Создание таблицы `subscribers` с полями id и email в БД. PHP-функция Subscribers() (файл function.php), вносящая emails подписчиков в созданную таблицу БД.
2 этап. Верстка электронного письма (файл mail_message.tpl) с размещением меток. PHP-функция Newsletter() (файл function.php), заменяющая метки на соответствующую информацию после момента добавления новости админом сайта и отправляет получившееся письмо через цикл на адреса подписчиков из БД.
